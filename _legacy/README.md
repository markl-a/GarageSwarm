# Legacy Code Archive

This directory contains archived code and documentation from the previous version of the Multi-Agent Platform.

## Contents

### /backend
The original FastAPI backend with complex features including:
- Evaluators (code quality, architecture, security, testability)
- Complex middleware stack
- Checkpoint and proposal systems
- Task templates and scheduling
- Extensive test suites

### /bmad
BMAD (BMAD Method) related files for AI agent development methodology.

### /old-docs
Original documentation including:
- Architecture deep dives
- API references
- Sprint plans
- UX design specifications
- Blog posts

### /old-docker
Previous Docker configurations including:
- Database init scripts
- Prometheus monitoring setup

### /old-root
Miscellaneous files from the project root:
- Old README and documentation
- Test scripts and configs
- CI/CD configurations
- Environment templates

### /old-scripts
Legacy utility scripts.

## Note

This code is preserved for reference only. The project has been rebuilt with a simplified architecture. See the main project README and ARCHITECTURE.md for the current implementation.
